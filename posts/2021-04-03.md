## What is flav-md?
### import flav-md

#### load from text
CommonJS
```js
const flavmd = require('flav-md');
const result = flavmd
  .createFlavMd()
  .readMdText('# hogehoge')
  .readCssText('.flav-md-h1 {color: red;}')
  .build();
```

ES6 or TypeScript
```ts
import {createFlavMd} from 'flav-md';

const result = createFlavMd()
  .readMdText('# hogehoge')
  .readCssText('.flav-md-h1 {color: red;}')
  .build();
```
